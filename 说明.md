# Google Research Football 本地开发说明

## 环境信息

- Python 虚拟环境：`D:\code\football-master\football-env`
- vcpkg 路径：`D:\code\vcpkg`
- Python 版本：3.10

---

## 日常使用

### 1. 激活虚拟环境

每次打开新的终端，先激活环境：

```ps
D:\code\football-master\football-env\Scripts\Activate.ps1
```

激活后命令行前面会出现 `(football-env)`。

### 2. 运行游戏

```bat
python -m gfootball.play_game --action_set=full
python -m gfootball.play_game --players=gamepad:left_players=1 --action_set=full
```

**按键说明：**

| 按键 | 功能 |
|------|------|
| 方向键 | 移动 |
| S | 短传 / 施压 |
| D | 射门 / 全队施压 |
| A | 高传 / 铲球 |
| W | 长传 / 门将出击 |
| Q | 换人 |
| C | 带球 |
| E | 冲刺 |
| **ESC** | **退出游戏** |

使用 **手柄** 时：与官方默认相比，face 键里对应键盘 **D / A** 的两颗（Pygame **按钮 1 / 2**）已对调——**1** = 高传/铲球，**2** = 射门/全队施压。

## 修改代码

### 修改 Python 代码（无需重新编译）

Python 代码位于 `gfootball/` 目录下，直接修改保存后**立即生效**，重新运行游戏即可看到效果。

常用文件：
- `gfootball/play_game.py` — 游戏主入口
- `gfootball/env/players/keyboard.py` — 键盘控制
- `gfootball/env/football_env.py` — 游戏环境逻辑

### 修改 C++ 引擎代码（需要重新编译）

C++ 引擎代码位于 `third_party/gfootball_engine/` 目录下。修改后需要重新编译：

**第一步：** 打开一个新的 PowerShell 或命令提示符（无需激活 venv）

**第二步：** 设置环境变量并编译

```bat
set VCPKG_ROOT=D:\code\vcpkg
set PY_VERSION=3.10
D:\code\football-master\football-env\Scripts\pip.exe install -e D:\code\football-master --no-build-isolation
```

```ps
$env:VCPKG_ROOT = "D:\code\vcpkg"
$env:PY_VERSION = "3.10"
$env:GENERATOR_PLATFORM = "x64"
& "D:\code\football-master\football-env\Scripts\pip.exe" install -e "D:\code\football-master" --no-build-isolation
```

> 注意：`set` 命令等号后面不要有空格，即 `set VCPKG_ROOT=D:\code\vcpkg` 而不是 `set VCPKG_ROOT = D:\code\vcpkg`

编译时间约 1~3 分钟（vcpkg 的依赖库已经缓存，只会重新编译引擎本身）。

编译完成后看到 `Successfully installed gfootball-2.10.3` 表示成功。

## 目录结构说明

```
football-master/
├── gfootball/              # Python 代码（直接改，立即生效）
│   ├── play_game.py        # 游戏入口
│   ├── env/                # 游戏环境

├── third_party/
│   └── gfootball_engine/   # C++ 引擎（改了需要重新编译）
├── football-env/           # Python 虚拟环境（不要手动改这里）
└── 说明.md                 # 本文件
```

## 常见问题

**Q：运行游戏时提示 `ModuleNotFoundError`**

先确认虚拟环境已激活，然后检查缺少哪个包并安装：
```bat
pip install <包名>
```

**Q：编译时提示找不到 `VCPKG_ROOT`**

确认 `set` 命令等号两边没有空格。

**Q：改了 C++ 代码但没效果**

需要重新执行上面的编译命令，不能只重新运行游戏。
